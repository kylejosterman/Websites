Ticket Manager

Ticket Manager is a class project developed in Java. A Ticket Manager system that uses a FSM modified from the ServiceNow incident management system state model. An employee who has a technical problem or who needs a new computer or new VM can make a request to the IT staff in the form of a ticket.

Pack Doption

Pack Doption is a class project developed in Java. It allows Animal rescue groups track adopted and available animals, and they remain in contact with owners after adoption for special events. Each rescue group works with a veterinarian, who sees animals for appointments stored as a queue.
